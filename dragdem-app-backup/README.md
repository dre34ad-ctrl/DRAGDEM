# DRAGDEM.com MVP

This is the MVP for DRAGDEM.com, the professional backbone for drag excellence.

## Features
- **Landing Page**: High-energy introduction to the platform.
- **Performer Dashboard**: Overview of revenue, bookings, and inquiries.
- **The Drag Vault**: Asset management for costumes, wigs, and media.
- **Digital Media Kit**: Professional public profile for performers.

## Tech Stack
- **Next.js 16** (App Router)
- **Tailwind CSS 4**
- **Lucide React** (Icons)
- **Next/Font** (Google Fonts: Montserrat, Inter, Playfair Display)

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Routes
- `/`: Landing Page
- `/dashboard`: Performer Dashboard
- `/vault`: The Drag Vault (Asset Management)
- `/profile/sasha-sparkle`: Public Performer Profile
